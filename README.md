# Teslo API NestJS

1. Clone the repository
2. ```yarn install```
3. Clone the ```.env.template``` file and rename it to ```.env```
4. Fill the env vars info
5. Build the database
```
docker-compose up -d
```
6. Start the app with
```
yarn start:dev
```
